# Financial Closeout Web Dashboard

A React-based web dashboard for viewing Procore-QuickBooks financial closeout reconciliation data.

## Overview

This web application provides a visual interface for viewing reconciliation reports generated by the CLI tool. It connects to a Supabase database to display:

- Project overview and status
- Reconciliation results by category
- Closeout items (action items) with status tracking
- AI-generated executive summaries

## Setup

### Prerequisites

- Node.js 20+
- Supabase account (free tier works)

### 1. Create Supabase Project

1. Go to [Supabase](https://supabase.com/) and create a new project
2. In the SQL Editor, run the migration script:
   - Open `supabase/migrations/001_initial_schema.sql`
   - Execute the entire script

### 2. Configure Environment

Create a `.env` file:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

Get these values from your Supabase project settings.

### 3. Install Dependencies

```bash
npm install
```

### 4. Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Deployment to Netlify

### Automatic Deployment

1. Connect your Git repository to Netlify
2. Configure build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
3. Add environment variables:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`

### Manual Deployment

```bash
npm run build
netlify deploy --prod --dir=dist
```

## Features

### Dashboard
- Overview of all projects
- Summary statistics (warnings, critical issues, exposure)
- Quick navigation to project details

### Project Detail
- Project information
- List of reconciliation reports
- Link to detailed report views

### Report Detail
- Key financial metrics
- Tabbed interface for different result categories:
  - Summary/Executive Summary
  - Commitments
  - Invoices
  - Change Orders
  - Retention
  - Budget vs Actual
  - Closeout Items

### Closeout Items
- Global view of all open action items
- Filter by status and category
- Update status (Open → In Progress → Resolved)
- Priority and amount at risk tracking

## Syncing Data from CLI

The CLI tool writes reconciliation data to JSON files. To sync this data to Supabase:

1. After running the CLI closeout command:
   ```bash
   python main.py closeout --project-id 12345 --output ./reports/
   ```

2. Use the sync script (or manually insert via Supabase dashboard):
   ```bash
   python scripts/sync_to_supabase.py ./data/reconciliation.json
   ```

A sync script can be created to parse the JSON output and insert into Supabase tables.
